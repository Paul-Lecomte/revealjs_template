# Reveal.js Template

A ready-to-use template to create reusable Reveal.js presentations.

## Contents

- `index.html`: main page with Reveal.js loaded via CDN.
- `slides.md`: slides written in Markdown.
- `css/custom.css`, `css/theme-overrides.css`: custom styles and theme overrides.
- `js/custom.js`: helpers and a simple timer widget.
- `print/print.css`: PDF-specific styles.
- `docs/USAGE.md`, `docs/TIPS.md`: usage guide and best practices.
- `assets/`: put your images, videos, and fonts here.

## Quick start

```powershell
npx serve . -l 8080
# then open http://localhost:8080
```

See `docs/USAGE.md` for more details, including the .NET option.
